# Credit Analysis Project

## Overview
This project demonstrates credit risk analysis for customers using DAX formulas and Power BI. It focuses on customer segmentation based on credit scores and other relevant attributes.

## Project Structure
- `Data/` - Contains the dataset used for analysis.
- `DAX/` - Contains DAX formulas for risk segmentation and calculations.
- `Scripts/` - Python or SQL scripts for data preprocessing and analysis.

## How to Use
1. Load the dataset into Power BI or Excel.
2. Use the DAX formulas from `DAX/` folder for analysis.
3. Optional: Run Python scripts in `Scripts/` folder for preprocessing.
